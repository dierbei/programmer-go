
# 程序员向前冲

面向社招程序员的一站式网站

## Dev

建议安装`git commit`提交规范辅助工具

```shell
# git 提交辅助工具 可选 参考 https://coding3min.com/1617.html
npm install -g conventional-changelog
npm install -g conventional-changelog-cli
```

使用 `git cz` 进行提交
